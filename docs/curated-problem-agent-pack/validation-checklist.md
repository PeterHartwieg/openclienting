# Validation Checklist

- SME status is evidenced
- organization website and identity are correct
- source URLs resolve
- all sources are in English
- evidence notes match the actual source
- all direct claims include field-level citation_refs
- problem description stays within what the source supports
- requirements are either directly supported or clearly marked as inference
- pilot framework is either directly supported or clearly marked as inference
- provider organization is present only when source-supported
- proposed tags have been resolved against the live taxonomy before import
- no duplicate organization exists in the target dataset
- packet is complete enough for import
